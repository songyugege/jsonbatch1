=>DTD           Vs                    XSD
DTD is base on SGML syntax              XSD are written in HTML
DTD support limited Data Type           XSD has wide range of datatype
DTD does not support Namespace           XSD support Namespace
DTD is not extensible                   XSD is extensible

=>Data Type
For number = CDATA ( Contain Character Data or Unprased Data)
For string = PCDATA ( Contain Prased Character Data)

=DTD + XML Document

<?xml version="1.0" ?>

    <!ELEMENT product(name,description,price,status)>
    <!ELEMENT name(#PCDATA)>
    <!ELEMENT description(#PCDATA)>
    <!ELEMENT price(#CDATA)>
    <!ELEMENT status(#PCDATA)>
]>
<product>
